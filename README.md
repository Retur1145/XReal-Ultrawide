## ❓ Troubleshooting

### The app will not open
- Check that the download finished
- Open the app again from Applications
- Right-click the app and choose Open if macOS blocks it
- Make sure your Mac allows apps from identified developers if needed

### The glasses are not detected
- Unplug the glasses
- Plug them in again
- Try a different USB-C port or adapter
- Restart the app
- Restart your Mac

### The display looks wrong
- Open macOS display settings
- Set the correct resolution
- Change scaling
- Reconnect the glasses
- Restart the app

### The menu bar icon does not appear
- Make sure the app is running
- Check if the icon is hidden behind other menu bar items
- Open the app again from Applications
